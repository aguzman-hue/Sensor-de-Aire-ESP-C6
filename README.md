# Sensor-de-Aire-ESP-C6
Este repositorio contiene los esquemáticos y el diseño de la placa  para un sensor de monitoreo ambiental basado en el SGP40. El sistema utiliza el microcontrolador ESP32-C6 para procesar y enviar datos. Viene en una placa circular de 4 capas, diseñada para ser utilizada en dispositivos portátiles o wearables.
