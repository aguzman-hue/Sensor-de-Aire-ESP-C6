# Sensor-Calidad-de-Aire

## Descripción del Proyecto
Este repositorio contiene el diseño completo de hardware (esquemático y PCB) para un nodo de monitoreo ambiental basado en el **SGP40** de Sensirion. El sistema utiliza el microcontrolador **ESP32-C6** para el procesamiento y transmisión de datos.

### Impacto y Aplicaciones:
Este proyecto ha sido diseñado con un formato circular compacto de 4 capas, lo que permite su integración en una amplia gama de dispositivos:

* **Wearables de Salud Ambiental:** Monitoreo personal de la calidad del aire en tiempo real para usuarios con condiciones respiratorias.
* **Ciudades Inteligentes (Smart Cities):** Despliegue de nodos distribuidos de bajo costo para el mapeo de la calidad del aire en oficinas, hogares o espacios públicos.
* **Sistemas de Seguridad Industrial:** Detección temprana de VOCs (Compuestos Orgánicos Volátiles) en entornos cerrados, mejorando los protocolos de ventilación automática.
* **IoT Portátil:** Gracias a su gestión eficiente de energía, es ideal para soluciones de monitoreo donde la autonomía de la batería es crítica.
